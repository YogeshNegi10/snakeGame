Snake Game is a classic arcade-style game built using HTML, CSS, and JavaScript. The player controls a snake that moves across a grid to eat food and grow longer. Each time the snake eats food, the score increases and the game becomes more challenging. The game ends if the snake collides with the wall or its own body.

The game includes smooth controls, real-time score tracking, increasing difficulty levels, and a high-score system for added competitiveness. A simple and responsive UI ensures an enjoyable experience on both desktop and mobile devices.

Features:

Classic snake movement using keyboard controls

Score and high-score tracking

Level progression with increasing speed

Lives system and game-over screen

Clean, responsive design

This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, game loops, and collision detection while recreating a nostalgic gaming experience.
